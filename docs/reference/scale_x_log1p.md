# A log1p x scale

A log1p x scale

## Usage

``` r
scale_x_log1p(..., n = 5, base = 10, dp = 0)
```

## Arguments

- ...:

  Other arguments passed on to `scale_(x|y)_continuous()`

- n:

  the number of major breaks

- base:

  the base for the logarithm

- dp:

  decimal points

## Value

a ggplot scale
